# AI PhD Application Assistant

这是仓库主页说明。当前只保留 **PhD 申请辅助项目**，项目代码在：

- `phd-outreach-copilot/`

---

## 快速开始

```bash
cd phd-outreach-copilot
npm install
npm run dev
```

打开：<http://localhost:3000>

---

## 项目功能（摘要）

- 导师检索与信息抽取（Discover）
- 匹配评分（Match Score）
- 套磁邮件生成（首封 + Follow-up）
- 申请管道看板（Dashboard）
- 导师详情与历史邮件线程
- Supabase 持久化（可选）
- 到期跟进提醒接口
